# Renaissance

Renaissance is a full-stack job matching platform designed to reduce bias in hiring through anonymized profiles and structured data processing.

## Tech Stack
- Frontend: React (Next.js, Tailwind CSS)
- Backend: Django + Django REST Framework
- Database: PostgreSQL
- Authentication: Firebase Authentication
- Ranking Engine: Java

## Features
- Anonymous candidate profiles
- Dual interface (Seeker / Recruiter)
- Swipe-based job matching
- Data anonymization pipeline

## Purpose
This project was developed as a senior capstone to explore bias reduction in hiring systems through software design.
